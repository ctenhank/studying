# Operators

각 연산자에 대한 자세한 설명은 다른 언어들과 동일하니 생략

### 산술연산자

`+`, `-`, `*`, `/`, `%`, `++`, `--`

### 관계연산자

`==`, `!=`,`>` `>=`, `<`, `<=`

### 논리연산자

`&&`, `||`, `!`

### 비트연산자

`&`, `|`, `~`, `<<`, `>>`

### 할당연산자

`=`, `+=`, `&=` ...

### 포인터 연산자

- `c/c++`과 동일하게 `&`, `*`를 사용하여 해당 변수의 주소를 얻거나 참조할때 사용
- 포인터 산술은 제공하지 않음
