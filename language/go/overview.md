# golang

- Go는 2009년 구글이 개발한 프로그래밍 언어
- Gabage collection과 Concurrency를 잘 지원해주는 **컴파일 언어**
- 컴파일러는 세 가지가 존재:
  - Google Go Toolchain
  - gccgo
  - GopherJS
- Go는 그리 좋지 않은 하드웨어에서도 빠르게 컴파일될 수 있도록 디자인

## Objective

- 안전성 : 타입 안전성과 메모리 안전성
- 병행성과 통신을 위한 훌륭한 지원
- 효과적인 가비지 컬렉션
- 빠른 컴파일
